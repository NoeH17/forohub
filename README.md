<h1>challenge-foro-hub</h1>

<h2>📝 Descripción del proyecto</h2>
Este proyecto consiste en la creación de un **Foro Hub**.  
El objetivo es construir una **API REST con Spring** que permita manejar los tópicos de un foro, donde los usuarios pueden hacer preguntas y recibir respuestas.

La idea es replicar cómo funciona un foro por detrás: cómo se guardan los datos, cómo se relacionan los tópicos con las respuestas y cómo los usuarios interactúan con ellos.  
Al final tendremos una API que implemente las operaciones básicas de un CRUD: **crear, mostrar, actualizar y eliminar tópicos**.

<h2>⚙️ Funcionalidades</h2>

- **Crear un nuevo tópico:** Permite que los usuarios publiquen sus preguntas.
- **Mostrar todos los tópicos:** Lista completa de los tópicos creados.
- **Mostrar un tópico específico:** Consultar un tópico en detalle.
- **Actualizar un tópico:** Editar la información de un tópico existente.
- **Eliminar un tópico:** Borrar un tópico de forma lógica o definitiva.
- **Validaciones:** Reglas de negocio para asegurar que los datos sean correctos.
- **Base de datos:** Persistencia de la información con migraciones.
- **Autenticación y autorización:** Control de acceso para proteger la información.

<h2>🔗 ¿Cómo pueden usarlo los usuarios?</h2>

Los usuarios podrán interactuar con la API a través de las rutas REST implementadas.  
Esto les permitirá crear y consultar tópicos, además de actualizarlos o eliminarlos según sea necesario.  
La API sigue buenas prácticas y está pensada para ser clara, segura y fácil de usar.
